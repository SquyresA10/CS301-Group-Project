C++ Operators

Understanding Operators in C++

Operators are special symbols that perform operations on variables and values. C++ provides a rich set of operators that allow you to perform mathematical calculations, compare values, assign values to variables, and control program flow.

Mastering operators is essential for writing efficient and functional C++ programs as they form the building blocks of any computational logic in your code.

Types of Operators in C++

C++ operators can be categorized into the following groups:

  1. Arithmetic Operators: Perform mathematical operations
  2. Assignment Operators: Assign values to variables
  3. Comparison Operators: Compare two values
  4. Logical Operators: Combine conditional statements
  5. Bitwise Operators: Perform bit-level operations
  6. Increment/Decrement Operators: Increase or decrease value by 1
  7. Other Operators: Special purpose operators

Arithmetic Operators

Arithmetic operators perform mathematical operations on numeric values:

Arithmetic Operators
Operator Name Description Example
+ Addition Adds two values x + y
- Subtraction Subtracts one value from another x - y
* Multiplication Multiplies two values x * y
/ Division Divides one value by another x / y
% Modulus Returns the division remainder x % y
Example: Arithmetic Operations
#include using namespace std; int main() { int a = 10; int b = 3; // Arithmetic operations int sum = a + b; // 13 int difference = a - b; // 7 int product = a * b; // 30 int quotient = a / b; // 3 (integer division) int remainder = a % b; // 1 // Display results cout << "a = " << a << ", b = " << b << endl; cout << "a + b = " << sum << endl; cout << "a - b = " << difference << endl; cout << "a * b = " << product << endl; cout << "a / b = " << quotient << endl; cout << "a % b = " << remainder << endl; // Float division float c = 10.0; float d = 3.0; float floatQuotient = c / d; // 3.33333 cout << "c / d = " << floatQuotient << endl; return 0; }
Output
a = 10, b = 3
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1
c / d = 3.33333

Assignment Operators

Assignment operators are used to assign values to variables. The basic assignment operator is =, but C++ also provides compound assignment operators that perform an operation and assignment in one step.

Assignment Operators
Operator Example Equivalent to
= x = 5 x = 5
+= x += 5 x = x + 5
-= x -= 5 x = x - 5
*= x *= 5 x = x * 5
/= x /= 5 x = x / 5
%= x %= 5 x = x % 5
Example: Assignment Operations
#include using namespace std; int main() { int x = 10; cout << "Initial value: x = " << x << endl; // Basic assignment x = 20; cout << "After x = 20: x = " << x << endl; // Compound assignments x += 5; // x = x + 5 cout << "After x += 5: x = " << x << endl; x -= 10; // x = x - 10 cout << "After x -= 10: x = " << x << endl; x *= 2; // x = x * 2 cout << "After x *= 2: x = " << x << endl; x /= 5; // x = x / 5 cout << "After x /= 5: x = " << x << endl; x %= 3; // x = x % 3 cout << "After x %= 3: x = " << x << endl; return 0; }
Output
Initial value: x = 10
After x = 20: x = 20
After x += 5: x = 25
After x -= 10: x = 15
After x *= 2: x = 30
After x /= 5: x = 6
After x %= 3: x = 0

Comparison Operators

Comparison operators are used to compare two values. They return a boolean result (true or false), which is essential for conditional statements and loops.
