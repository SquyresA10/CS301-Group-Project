üìù What are Variables?

Variables are containers that store data values. In C++, every variable has a specific type that determines what kind of data it can hold and how much memory it occupies. Think of variables as labeled boxes where you can store different types of information.

üí° Key Concept

A variable in C++ must be declared with a specific data type before it can be used. This tells the compiler how much memory to allocate and what operations are allowed on that variable.

üîß Declaring Variables

To declare a variable in C++, you need to specify the data type followed by the variable name. You can also initialize the variable with a value at the time of declaration.

Variable Declaration Syntax 
// Basic variable declaration
int age;                    // Declaration only
int score = 95;            // Declaration with initialization

// Multiple variables of same type
int x, y, z;              // Multiple declarations
int a = 10, b = 20, c = 30; // Multiple with initialization
‚ö†Ô∏è Important Rule

In C++, you must declare a variable before using it. Using an uninitialized variable can lead to unpredictable behavior!

üè∑Ô∏è Common Variable Types

C++ provides several built-in data types for different kinds of data. Here are the most commonly used ones:

Common Data Types 
// Integer types
int studentCount = 25;        // Whole numbers
long population = 7800000000;  // Large whole numbers

// Floating-point types
float temperature = 98.6f;     // Single precision decimal
double pi = 3.14159265359;    // Double precision decimal

// Character and string types
char grade = 'A';             // Single character
string name = "John Doe";      // Text string

// Boolean type
bool isStudent = true;       // True or false

üîí Constants

Constants are variables whose values cannot be changed after they are initialized. They are useful for storing values that should remain the same throughout your program, like mathematical constants or configuration values.

Declaring Constants 
// Using const keyword
const double PI = 3.14159;
const int MAX_STUDENTS = 30;
const string SCHOOL_NAME = "CodingCompanion Academy";

// Using #define preprocessor directive
#define SPEED_OF_LIGHT 299792458

// Constants must be initialized when declared
const int DAYS_IN_WEEK = 7;  // ‚úì Correct
// const int MONTHS_IN_YEAR;     // ‚úó Error: must initialize
üö® Remember

Constants must be initialized when they are declared, and their values cannot be modified later in the program. Attempting to change a constant's value will result in a compilation error.

üìã Variable Naming Rules

C++ has specific rules for naming variables. Following these rules and conventions makes your code more readable and maintainable:

Valid and Invalid Variable Names 
// Valid variable names
int age;              // Simple name
int student_count;   // Using underscore
int maxScore;        // CamelCase
int _private;        // Starting with underscore
int value2;          // Ending with number

// Invalid variable names
// int 2value;           // Cannot start with number
// int student-count;    // Cannot use hyphen
// int class;            // Cannot use keywords
// int my variable;      // Cannot have spaces
üí° Best Practices
  • Use descriptive names: studentAge instead of a
  • Use camelCase for variables: firstName, totalScore
  • Use UPPER_CASE for constants: MAX_SIZE, PI
  • Avoid using single letters except for loop counters
