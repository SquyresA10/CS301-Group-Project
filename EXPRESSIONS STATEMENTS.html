📝 Control Flow Basics

Control flow statements allow your program to make decisions and execute different code blocks based on conditions. In C++, the primary control flow statements are if, if-else, and switch. These statements evaluate expressions and determine which code path to follow.

💡 Key Concept

An expression in C++ is any combination of literals, variables, operators, and function calls that evaluates to a single value. Statements are complete lines of code that perform actions, including control flow statements.

🔧 The if Statement

The if statement is the simplest form of conditional execution. It evaluates a condition in parentheses, and if the condition is true, the code block following the statement is executed.

Basic if Statement 
int age = 18;

if (age >= 18) {
    cout << "You are an adult." << endl;
}
⚠️ Important Note

The condition in an if statement must evaluate to a boolean value (true or false). In C++, any non-zero value is considered true, and zero is considered false.

🔄 if-else Statements

The if-else statement provides an alternative path of execution when the condition is false. You can also chain multiple conditions using else if.

if-else Examples 
// Basic if-else
int score = 85;

if (score >= 90) {
    cout << "Grade: A" << endl;
} else {
    cout << "Grade: Not A" << endl;
}

// Multiple conditions with else if
if (score >= 90) {
    cout << "A" << endl;
} else if (score >= 80) {
    cout << "B" << endl;
} else if (score >= 70) {
    cout << "C" << endl;
} else {
    cout << "F" << endl;
}

🎯 Nested if Statements

You can place if statements inside other if statements to create more complex decision-making structures. This is called nesting.

Nested if Example 
int age = 25;
bool hasLicense = true;

if (age >= 18) {
    cout << "You are old enough to drive." << endl;
    
    if (hasLicense) {
        cout << "You can legally drive!" << endl;
    } else {
        cout << "But you need a license first." << endl;
    }
} else {
    cout << "You are too young to drive." << endl;
}
💡 Best Practice

While nesting can be powerful, deep nesting (more than 2-3 levels) can make code hard to read. Consider using functions or logical operators to simplify complex conditions.

🔀 switch Statements

The switch statement provides an alternative to long if-else chains when you need to compare a variable against multiple constant values. It's often more readable for such cases.

switch Statement Example 
char grade = 'B';

switch (grade) {
    case 'A':
        cout << "Excellent!" << endl;
        break;
    case 'B':
        cout << "Good job!" << endl;
        break;
    case 'C':
        cout << "Fair." << endl;
        break;
    case 'D':
        cout << "Needs improvement." << endl;
        break;
    default:
        cout << "Invalid grade." << endl;
}
🚨 Remember

Always include break statements at the end of each case block in a switch statement. Without them, execution will "fall through" to the next case, which is rarely what you want (unless intentionally designed that way).

❓ Ternary Operator

C++ provides a shorthand conditional operator (?:) that can be used as a concise alternative to simple if-else statements.

Ternary Operator Examples 
// Basic ternary operator
int age = 20;
string status = (age >= 18) ? "Adult" : "Minor";

// Equivalent to:
string status;
if (age >= 18) {
    status = "Adult";
} else {
    status = "Minor";
}
⚠️ Caution

While the ternary operator can make code more concise, overusing it or nesting multiple ternary operators can make your code harder to read. Use it judiciously for simple conditions.
