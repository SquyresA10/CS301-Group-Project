C++ Syntax

Introduction to C++ Syntax

C++ syntax refers to the set of rules that define how C++ programs are written and interpreted. Learning the correct syntax is essential to writing functional code. In this lesson, we'll cover the basic structure of a C++ program and the fundamental syntax rules you need to know.

Basic Structure of a C++ Program

Every C++ program follows a specific structure. At its most basic level, a C++ program consists of:

C++
#include using namespace std; int main() { // Your code goes here cout << "Hello, World!"; return 0; }

Let's break down this program structure:

  • #include <iostream>: This is a preprocessor directive that includes the input/output stream library.
  • using namespace std;: This statement allows us to use names for objects and variables from the standard library.
  • int main(): The main function is the entry point of any C++ program. Execution begins from this function.
  • { }: Curly braces define a block of code.
  • cout << "Hello, World!";: This statement outputs "Hello, World!" to the console.
  • return 0;: This statement terminates the main function and returns a value (0) to the operating system.

Statements and Semicolons

In C++, statements are instructions that perform specific actions. Each statement must end with a semicolon (;).

Example: C++ Statements
int x = 5; // Declaration and initialization x = x + 10; // Assignment cout << x; // Output return 0; // Return statement

Forgetting a semicolon is a common syntax error that will prevent your program from compiling.

C++ Case Sensitivity

C++ is case-sensitive, which means uppercase and lowercase letters are considered different characters. For example, myVariable, myvariable, and MYVARIABLE would be three different variables.

Example: Case Sensitivity
int age = 25; // Declares a variable named "age" int AGE = 30; // Declares a different variable named "AGE" cout << age; // Outputs 25 cout << AGE; // Outputs 30

Whitespace in C++

Whitespace refers to spaces, tabs, and newlines in your code. C++ generally ignores whitespace, which means you can format your code for readability without affecting its functionality.

Example: Whitespace Usage
// These two statements are equivalent int sum=x+y; int sum = x + y; // These blocks are also equivalent if(x>5){cout<<"Greater than 5";} if (x > 5) { cout << "Greater than 5"; }

While the compiler ignores most whitespace, using it properly makes your code more readable and maintainable.

Identifiers

Identifiers are names used to identify variables, functions, classes, or other entities in your program. C++ has specific rules for creating valid identifiers:

  • Must begin with a letter (a-z, A-Z) or underscore (_)
  • Can contain letters, digits (0-9), and underscores
  • Cannot contain spaces or special characters
  • Cannot be a reserved keyword
Example: Valid and Invalid Identifiers
// Valid identifiers int age; float _value; string firstName; double price2; // Invalid identifiers int 2value; // Cannot start with a number float user-name; // Cannot contain hyphens char for; // Cannot use reserved keywords
Key Takeaways
  • Every C++ program must have a main() function as its entry point.
  • Statements in C++ must end with a semicolon (;).
  • C++ is case-sensitive, so variable and Variable are different.
  • Whitespace is generally ignored by the compiler but improves code readability.
  • Identifiers must follow specific naming rules and cannot use reserved keywords.

  2. Write a simple C++ program that creates two variables and performs a calculation on them.
  3. List five valid and five invalid identifier names, and explain why each is valid or invalid.

  1. Declares three different variables with appropriate names
  2. Performs at least two different operations with these variables
  3. Displays the results using cout
  4. Uses proper indentation and formatting for readability
